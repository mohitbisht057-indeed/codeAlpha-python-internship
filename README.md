# CodeAlpha Python Internship

This repository contains two Python projects completed as part of the CodeAlpha Python Internship. These projects demonstrate core Python concepts such as loops, conditional statements, dictionaries, functions, user input, and file handling.

## Projects

### 1. Basic Chatbot
A rule-based chatbot that responds to predefined user inputs.

#### Features
- Greets the user
- Answers simple questions
- Handles unknown inputs
- Exits on user command

#### Technologies Used
- Python

---

### 2. Stock Portfolio Tracker
A Python application that calculates the total investment value based on stock prices and quantities.

#### Features
- Uses dictionaries to store stock prices
- Accepts user input
- Calculates total investment value
- Saves results to a text file

#### Technologies Used
- Python
